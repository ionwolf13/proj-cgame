### PROJ CGame
  Create an account and see the multiple card games that you can play! This application contains games like War, Go Fish,
  Black Jack, Spoon and many more. Come and play!

    - Flexbox <br>
    - Axios.js <br>
    - Node.js <br>
    
### Back End
  Utilized Ruby Back End <br>
    - JWT <br>
    - Bcrypt
